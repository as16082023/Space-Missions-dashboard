Take a visual trip through history and discover the incredible milestones of space exploration in a fun and easy way!"

A Comprehensive dashboard highlighting the captivating History of Space Missions using Tableau and it highlighting the key trends, patterns and actionable insights.

Key Objectives:
1. A visual representation showing the top countries with most launches
2. Analyzing year wise success vs failure
3. Discovering the leading company in space mission in each decade
4. A comparison between active and retired rockets in each decade
5. Showing the total number of companies participating in space mission
6. An overall count of space missions
7. A display of distinct type of rockets used in space missions
